# DULU DULU Website

This repository contains the DULU DULU restaurant website.

## Getting started

### Install dependencies

```bash
npm install
```

### Run locally

```bash
npm run dev
```

### Build

```bash
npm run build
npm run preview
```

## Notes

- Update images and menu assets in `public/` and `src/` as needed.
- Social preview uses `public/og-image.png`.
